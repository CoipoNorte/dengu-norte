# 🧠 Dengu Norte

**ATS Optimizer** — Plataforma profesional para optimizar currículums contra sistemas ATS.

## ✨ Características

- **Subir CV** (PDF, DOCX, DOC) con análisis automático ATS
- **Puntuación ATS** en tiempo real
- **Comparar con Oferta Laboral** — Mide tu compatibilidad
- **Generar CV ATS** — Crea una versión optimizada
- **Editor ATS 100%** — Edita y mejora tu CV
- **Dashboard** con estadísticas y progreso
- **100% Privado** — Todo se guarda en tu navegador (IndexedDB)

## 🛠️ Tecnologías

- React 19 + TypeScript
- Vite
- Tailwind CSS
- IndexedDB
- Lucide Icons + Sonner
